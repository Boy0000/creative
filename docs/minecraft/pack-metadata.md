## Resource Pack Metadata

Defined at the root `pack.mcmeta` file, supports `pack`and `language` 
metadata parts (See [Metadata](metadata.md))

### Example
```json
{
    "pack": {
        "pack_format": 8,
        "description": "My resource pack"
    },
    "language": {
        "custom": {
            "name": "Custom",
            "region": "Custom",
            "bidirectional": false
        }
    }
}
```
